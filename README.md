# The Fountain of Objects
This is a 'boss battle' challenge from the book *The C# Player's Guide* by RB Whitaker.
[https://csharpplayersguide.com/](https://csharpplayersguide.com/)


An exerpt from the book:
*The Fountain of Objects is a 2D grid-based world full of rooms. Most rooms are empty, but a few are unique rooms. One Room is the cavern entrance. Another is the fountain room, containing the Fountain of Objects.*
